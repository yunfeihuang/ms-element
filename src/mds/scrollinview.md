### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| effect     | 动画效果   | String  |   -       |    fadein    |
| duration     | transition duration   | String  |   -       |    1.3s    |
| delay    | transition delay   | String  |   -       |    0.2s    |
| direction     | 动画方向   | String  |   top,bottom,left,right  |    top    |
